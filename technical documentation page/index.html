Introduction

Python is a high-level, interpreted programming language known for its readability and flexibility.

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python has a large standard library and an active community that contributes a wealth of third-party modules.

This documentation serves as a brief overview of essential Python programming concepts.

Whether you’re a beginner or seasoned developer, Python’s simplicity makes it a great choice for many applications.

print("Hello, Python!")
Syntax and Indentation

Python uses indentation instead of braces to define blocks of code.

Proper indentation is not just for style—it's syntactically required in Python.

Typically, each block is indented with 4 spaces.

Incorrect indentation will result in an `IndentationError`.

This helps make Python code visually uncluttered and easy to follow.

if 5 > 2: print("Five is greater than two!")
  • Consistent indentation is mandatory
  • No need for semicolons
  • Colons (`:`) start code blocks
Data Types

Python provides various built-in data types.

Common types include: integers, floats, strings, lists, tuples, dictionaries, and booleans.

Variables in Python do not need explicit declaration to reserve memory space.

The data type is automatically determined from the value assigned.

You can check a variable's type using the `type()` function.

x = 5 print(type(x)) #
  • int: Integer numbers
  • float: Decimal numbers
  • str: Text
  • list, tuple, dict: Collections
  • bool: True or False
Control Structures

Python supports typical control structures such as `if`, `for`, and `while`.

Control flow is intuitive and relies on indentation to structure code blocks.

`if-elif-else` statements are used for decision making.

Loops are used for iteration.

You can break out of loops using `break`, or skip an iteration using `continue`.

for i in range(5): print(i)
  • `if` for conditional checks
  • `for` for iteration over sequences
  • `while` for condition-based loops
Functions

Functions in Python are defined using the `def` keyword.

They help break programs into smaller, modular chunks.

Functions can accept parameters and return values.

You can also use lambda functions for short anonymous functions.

Python also supports built-in functions like `len()`, `max()`, `sum()`, etc.

def greet(name): return "Hello " + name
  • Use `def` to define functions
  • Use `return` to return values
  • Use `lambda` for short one-line functions
